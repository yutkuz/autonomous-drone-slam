# Documentation Assets

This folder contains small documentation images that are safe to commit.

## Folders

- `gazebo/`: Gazebo and RViz screenshots used to explain the simulation environment.
- `outputs/`: Selected lightweight PNG/JPEG previews from SLAM outputs.
- `web-ui/`: Screenshots of the WebDemo interface.

## Not Included

The repository does not include full RTAB-Map databases, PLY point clouds, GLB/OBJ meshes or raw RGB/depth frame datasets. Those files are generated artifacts and are intentionally ignored by `.gitignore`.

## Naming

Image names are descriptive and stable so they can be referenced from README, reports and presentation material without relying on WhatsApp export filenames.
